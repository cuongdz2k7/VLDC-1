# materials

Thư mục này dùng để đặt tài liệu thầy gửi, ví dụ PDF ôn thi, slide bài giảng hoặc bài tập trọng tâm.

Quy tắc dùng tài liệu:

- Không sửa trực tiếp file PDF gốc.
- Khi hỏi AI, hãy nêu tên file tài liệu cần dùng.
- Nếu tài liệu có công thức hoặc kỹ thuật riêng của thầy, ưu tiên dùng theo tài liệu đó.
- Nếu trích ý từ tài liệu, hãy ghi nguồn để dễ kiểm tra lại.

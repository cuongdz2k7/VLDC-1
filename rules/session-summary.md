# Rule tổng kết sau buổi học

Dùng khi người học muốn tổng kết một buổi học cụ thể.

## File cần đọc thêm

- `SESSION_TEMPLATE.md`
- `PROFILE.md`
- File `exercises/` hoặc `topics/` liên quan nếu có.

## Việc cần làm

1. Tóm tắt nội dung buổi học.
2. Ghi phần đã biết và phần chưa chắc.
3. Ghi công thức chính trong buổi học.
4. Nếu có file tổng hợp kiến thức thực hành, ghi bài đã chọn, bài bỏ qua và lý do.
5. Ghi bài tập đã làm, kết quả và lỗi sai.
6. Ghi câu hỏi cần hỏi lại.
7. Ghi việc cần ôn tiếp.
8. Tự đánh giá: nhận ra dạng bài chưa, tự chọn công thức chưa, có kiểm tra đơn vị và dấu không.

## Khi cập nhật file

- Dùng `SESSION_TEMPLATE.md` làm cấu trúc.
- Không dùng file này để tổng kết cả chương.
- Chỉ cập nhật `MISTAKES.md`, `FORMULAS.md`, `QUESTIONS.md`, `LEARNING_PLAN.md` khi `Update` cho phép.

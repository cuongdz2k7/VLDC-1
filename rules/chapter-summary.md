# Rule tổng kết chương

Dùng khi người học đã học xong hoặc gần xong một chương và muốn tổng kết để ôn thi.

## File cần đọc thêm

- `PROFILE.md`
- `EXAM_CONTEXT.md`
- `EXAM_PATTERNS.md`
- `FORMULAS.md`
- `topics/README.md`
- File `exercises/[ten-chuong]-priority.md` nếu người học inject hoặc nếu file tồn tại.
- `LEARNING_PLAN.md` nếu cần xét lại tiến độ hoặc ưu tiên ôn tập.

## Việc cần làm

1. Đọc `topics/README.md` để lấy cấu trúc topic.
2. Đọc file `exercises/[ten-chuong]-priority.md` nếu người học inject hoặc nếu file tồn tại.
3. Xác định vai trò chương trong đề thi.
4. Rút kiến thức cốt lõi từ file thực hành, chỉ giữ phần cần để làm bài.
5. Ghi công thức trọng tâm và điều kiện áp dụng.
6. Lấy công thức bắt buộc liên kết với bài tập ưu tiên từ file `exercises/` nếu có.
7. Cô đọng dạng bài hay ra, dấu hiệu nhận diện trong đề, công thức lõi và quy trình giải chung từ khung tư duy theo dạng bài.
8. Ghi lỗi sai thường gặp.
9. Chọn bài đại diện nên xem lại, ưu tiên bài bắt buộc và bài nên làm.
10. Ghi câu hỏi còn mở nếu có.

## Nguyên tắc cô đọng từ `exercises/`

- Không copy nguyên bảng bài tập hoặc toàn bộ khung tư duy từ file `exercises/`.
- Topic phải ngắn hơn file thực hành và phục vụ ôn thi nhanh.
- Nếu file `exercises/` có mục `Công thức cần nhớ bắt buộc`, dùng mục đó làm nguồn chính cho công thức trong topic.
- Nếu file `exercises/` có mục `Khung tư duy theo dạng bài`, rút thành dạng bài hay ra, dấu hiệu nhận diện trong đề, công thức bật ra đầu tiên và quy trình giải chung.
- Mỗi dạng bài trong topic nên trả lời ngắn gọn: nhìn đề thấy gì, dùng công thức lõi nào, tư duy bước đầu ra sao.
- Topic không cần ghi toàn bộ `Chọn công thức tiếp theo theo câu hỏi đề` như priority, nhưng phải giữ các nhánh quan trọng nhất nếu dễ nhầm hoặc hay ra thi.
- Nếu thiếu file `exercises/`, vẫn có thể tổng kết từ input của người học, nhưng phải nói rõ nguồn thiếu.

## Khi cập nhật file

- Chỉ tạo hoặc cập nhật `topics/[ten-chuong].md` khi `Update` cho phép.
- Làm theo cấu trúc trong `topics/README.md`.
- Không dùng `SESSION_TEMPLATE.md` để tổng kết cả chương.
- Sau khi cập nhật topic, mới xét có cần cập nhật `FORMULAS.md`, `MISTAKES.md`, `QUESTIONS.md`, `LEARNING_PLAN.md` không.
- Nếu có lỗi/công thức/câu hỏi quan trọng, chỉ cập nhật `MISTAKES.md`, `FORMULAS.md`, `QUESTIONS.md` khi được phép rõ.
- Nếu topic cho thấy tiến độ hoặc thứ tự ôn tập cần đổi, chỉ cập nhật `LEARNING_PLAN.md` khi được phép rõ.

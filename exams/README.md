# exams

Thư mục này dùng để lưu đề cũ, đề thử và phân tích đề.

Gợi ý các file nên tạo sau:

- `de-01.md`: chép lại hoặc tóm tắt đề số 01.
- `de-02.md`: chép lại hoặc tóm tắt đề số 02.
- `mock-exam-01.md`: đề luyện mô phỏng.
- `mistakes-analysis.md`: phân tích lỗi sau khi làm đề.

Mỗi đề nên ghi:

- Thời gian làm bài.
- Cấu trúc điểm.
- Phạm vi kiến thức.
- Đề bài.
- Phân tích chương, dạng bài và mức độ.
